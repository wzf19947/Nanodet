## 环境配置
参考README.md配置环境即可

## 验证demo
测试图片：
```
python /demo.py
```

## 导出onnx
修改nanodet/model/head/nanodet_plus_head.py中的forward部分导出想要的onnx格式:
```
python export_onnx.py
```

## 训练模型：
```
python train.py
```

## 部署模型：
### python
```
python3 QRCode_onnx_infer_Nanodet.py
or
python3QRCode_axmodel_infer_Nanodet.py
```
### C++:
```
./ax_nanodetplus_qrcode_batch -m nanodet-plus-m_416_QR.axmodel -i qrcode_test/
```

## 备注
```
自训练模型导出onnx大小4569kb，u8量化后axmodel仍有2.39MB，比yolov5n(1.9MB)要大。在尝试删除aux_head、aux_fpn等训练中才有的节点后，模型并没有变小，结果也接近。但ax650上实际量化后cmm size比yolov5n小不少，耗时会略高。
```
